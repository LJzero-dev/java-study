jspSite 프로젝트
-1. 인클루드 파일 : /_inc/
 - inc_head.jsp : 모든 파일의 가장 위쪽에 넣을 파일로, 각종 import 및 DB연결, 로그인 여부, 공용으로 사용할 css와 js 파일등을 처리함(공용으로 사용할 메소드등도 정의할 수 있음)
 - inc_foot.jsp : 연결된 DB Connection 객체 닫기(close())

0. 인덱스 화면 : /index.jsp
 - 로그인/로그아웃 링크
 - 공지사항 링크
 - 자유게시판 링크


1. 로그인, 로그아웃
 - 로그인 폼 : /login_form.jsp
	로그인에 필요한 아이디와 암호를 받아 처리파일로 보냄
	사용자가 입력한 아이디와 암호에 대한 검사 (입력 여부와 글자 개수 검사 - 자바스크립트)
	로그인이 필요한 화면 접속시 로그인 폼으로 이동한 후 로그인후에 다시 해당 화면으로 이동시키기

 - 로그인 처리 : /login_proc.jsp
 	로그인 폼에서 보낸 아이디와 암호, 이동할 url 값을 받음
	받은 아이디와 암호를 이용하여 로그인 처리
	 - 아이디와 암호를 실제 회원테이블에서 검색한 후 결과를 리턴
	 - 로그인 성공 : 받아온 데이터들로 MemberInfo클래스의 인스턴스로 생성한 후 세션에 저장
	 				이동할 url이 있으면 그 곳으로 없으면 index로 이동시킴
	 - 로그인 실패 : 로그인 폼으로 이동시킨 후 "아이디 또는 비밀번호를 잘못 입력했습니다. 입력하신 내용을 다시 확인해주세요." 메시지를 보여줌
	 				이동할 url이 있으면 그 값도 로그인 폼으로 이동시 같이 보내줌

 - 로그아웃 : /logout.jsp
 	세션에 저장된 로그인 관련 정보를 삭제 시키고, index로 이동시킴


2. 공지사항
 - 공지글 목록화면 : /bbs/notice_list.jsp
	공지글의 목록 보여주기 : 페이지 번호에 해당하는 공지글들만 추출
	공지글 검색 : 제목, 내용, 제목+내용
	공지글 링크 : 공지글 보기 화면으로 특정 공지글을 연결
	공지글 등록 : 공지글 등록 화면으로 연결
	페이징 : 게시글들을 페이지로 구분하여 링크

 - 공지글 보기화면 : /bbs/notice_view.jsp
 	받아온 글번호에 해당하는 공지글을 보여줌
	조회수 증가
	수정폼으로의 링크 : 작성자에 한해서
	삭제기능으로의 링크 : 작성자에 한해서 컨펌창으로 물어본 후 연결

 - 공지글 등록/수정 폼 : /bbs/notice_form.jsp
 	공지글에 대한 등록 및 수정을 할 수 있는 폼 제공 : 수정시 기존 공지글이 나타남

 - 공지글 등록 처리 : /bbs/notice_proc_in.jsp
 	작성자가 입력한 데이터들을 받아 DB에 저장 후 입력한 글을 보러 보기화면으로 이동
	입력할 글번호는 기존의 가장 큰 번호에 1을 더한 값으로  새롭게 생성해야 함

 - 공지글 수정 처리 : /bbs/notice_proc_up.jsp
 	작성자가 수정한 데이터들을 받아 DB에 저장 후 수정한 글을 보러 보기화면으로 이동 	

 - 공지글 삭제 처리 : /bbs/notice_proc_del.jsp
 	작성자가 선택한 공지글을 DB에서 삭제 후 공지글 목록 1페이지로 이동


3. 자유게시판
 - 자유게시판 목록화면 : /bbs/free_list.jsp
	게시글의 목록 보여주기 : 페이지 번호에 해당하는 게시글들만 추출
	게시글 검색 : 제목, 내용, 제목+내용
	게시글 링크 : 게시글 보기 화면으로 특정 공지글을 연결
	게시글 등록 : 게시글 등록 화면으로 연결 - 비회원도 가능
	페이징 : 게시글들을 페이지로 구분하여 링크

 - 자유게시판 보기화면 : /bbs/free_view.jsp
 	받아온 글번호에 해당하는 공지글을 보여줌
	조회수 증가
	수정 링크 : 회원글은 작성자일 경우 수정폼으로, 비회원글은 비밀번호 입력폼으로 링크
	삭제 링크 : 회원글은 작성자일 경우 컨펌창으로 물어본 후 
	
 - 자유게시판 등록/수정 폼 : /bbs/free_form.jsp
 	게시글에 등록 폼 : 비회원일 경우 작성자와 비밀번호를 받을 수 있는 컨트롤을 보여줌
	게시글에 수정 폼 : 수정 시 작성자 여부 검사 후 기존 게시글을 보여줌

 - 자유게시판 비밀번호 입력 폼 : /bbs/free_form_pwe.jsp
	비회원글을 수정 및 삭제 시 필요한 비밀번호를 입력받기 위한 폼
	수정 시 비밀번호 입력 후 수정 폼(free_form.jsp)으로 이동
	삭제 시 비밀번호 입력 후 삭제 처리(free_proc_del.jsp)로 이동

 - 자유게시판 등록 처리 : /bbs/free_proc_in.jsp
 	작성자가 입력한 데이터들을 받아 DB에 저장 후 입력한 글을 보러 보기화면으로 이동
	입력할 글번호는 기존의 가장 큰 번호에 1을 더한 값으로  새롭게 생성해야 함

 - 자유게시판 수정 처리 : /bbs/free_proc_up.jsp
 	작성자가 수정한 데이터들을 받아 DB에 저장 후 수정한 글을 보러 보기화면으로 이동 	

 - 자유게시판 삭제 처리 : /bbs/free_proc_del.jsp
 	작성자가 선택한 공지글을 DB에서 삭제 후 공지글 목록 1페이지로 이동
